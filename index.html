Game -- Who Wants to Be a Millionaire?

Welcome to the free online version of the award-winning game show Who Wants to Be a Millionaire?.

The game consits of 15 questions.divided into three sections(easy,medium and difficult),each section containing 5 questions and a time of 60seconds per question.

There are 3 Safe zones.

1.The $1000.00 question---

If you get this question wrong, you leave with nothing. If this question is answered correctly, you are guaranteed $1,000 even if you answer incorrectly before reaching the next safe Zone.

2.The $32,000.00 question.

If you get this question wrong, you leave witbh $1,000.00. If this question is answered correctly, you are guaranteed $32,000.00 even if the answer incorrectly before reaching the next safe Zone.

3.The $1,000,000.00 question.

Lifelines

You are allowed three Lifelines which you can use at any point during the game. Each Lifeline can only be used once.

50/50 — removes two wrong answers from the multiple-choice selection, leaving the you with only one correct and one incorrect option. This means you have a 50/50 chance.

Ask the Audience — the audience is asked the same question you were asked and a quick poll is done to show their answers.
